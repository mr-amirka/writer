# FEEDBACK — Отклонённые варианты

Этот файл заполняет агент при каждом отклонении варианта автором.
Цель — не повторять одни и те же ошибки в следующих вариантах.

---

## Формат записи

```
### [Дата] · [Что отклонено]
**Причина:** [что именно не понравилось]
**Вывод для агента:** [чего избегать в следующий раз]
```
