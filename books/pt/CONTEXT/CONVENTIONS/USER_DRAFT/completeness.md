# Requisitos da Autora: Completude — Notas em Bruto

*Input em bruto da autora. O agente lê como material de partida.*
*Escrito: 2026-06-06*

---

O agente por vezes para antes de uma tarefa estar realmente concluída. Exemplo: pedi para duplicar a estrutura do domínio russo (com livro demo) para inglês e português. O agente criou o template mas não criou livros demo reais — só mais tarde, quando perguntei directamente, é que ficou claro que faltavam. Tinha "terminado" sem terminar.

Quero:
- Antes de o agente dizer "concluído" — que verifique mesmo se está concluído.
- Se uma tarefa implica criar N ficheiros — que os conte.
- Se falta alguma coisa — que o diga. Não que espere que eu não repare.
- Se uma sessão termina a meio de uma tarefa — o STATUS.md deve reflectir o que está realmente feito.
- Ao duplicar estrutura de um domínio para outro — comparar com o original. Não por intuição, por contagem.

Não devo ter de verificar cada tarefa para saber se foi concluída.
